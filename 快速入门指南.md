# NextMD 快速入门指南

## 安装步骤

1. **准备Python环境**
   - 确保您的系统已安装Python 3.6或更高版本
   - 可以在命令行中输入 `python --version` 检查版本

2. **安装依赖**
   ```
   pip install markdown mistune html2text
   ```

3. **运行程序**
   ```
   python main.py
   ```

## 基本操作

### 文件操作
- **新建**：`Ctrl+N`
- **打开**：`Ctrl+O`
- **保存**：`Ctrl+S`
- **另存为**：`Ctrl+Shift+S`

### 编辑操作
- **撤销**：`Ctrl+Z`
- **重做**：`Ctrl+Y`
- **查找**：`Ctrl+F`
- **替换**：`Ctrl+H`

### 格式转换
- Markdown → HTML：菜单「转换」→「Markdown转HTML」
- HTML → Markdown：菜单「转换」→「HTML转Markdown」

## 配置说明

### 方式一：环境变量文件（推荐）
1. 复制 `.env.example` 文件并重命名为 `.env`
2. 编辑 `.env` 文件，修改以下配置：
   ```
   HOST=localhost  # 部署地址
   PORT=3367       # 部署端口
   ```

### 方式二：命令行参数
```
python main.py --host 127.0.0.1 --port 8080
```

## 拖放支持
- 直接将 `.md` 或 `.html` 文件拖放到程序窗口即可打开

## 常见问题

**问题**：程序启动失败
**解决方法**：检查Python版本和依赖是否正确安装

**问题**：文件无法打开
**解决方法**：确保文件编码为UTF-8，检查文件权限

**问题**：配置不生效
**解决方法**：确保 `.env` 文件位于正确位置，检查格式是否正确

## 日志位置
- 日志文件位于 `logs` 文件夹下
- 文件名格式：`nextmd_YYYYMMDD.log`
